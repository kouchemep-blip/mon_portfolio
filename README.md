## ⚙️ Installation locale

Clonez le dépôt :

```bash
git clone https://github.com/votre-username/votre-repo.git
```

Accédez au dossier :

```bash
cd votre-repo
```

Installez les dépendances :

```bash
npm install
```

Lancez le serveur de développement :

```bash
npm run dev
```

Construisez la version de production :

```bash
npm run build
```
